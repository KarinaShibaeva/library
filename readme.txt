Возможности пользователя:

На стартовой странице расположен логотип сайта, название, лозунг, кнопка "Каталог книг".
При нажатии на кнопку "Каталог книг" осуществляется перенос на соответсвующую страницу.

На странице "Каталог книг" расположены карточки с названиями и автором. При нажатии на "Показать еще" выполняется переход на страницу с книгами данной категории.
При нажатии на книгу переход на страницу с полным описанием книги и кнопкой "Забронировать".
При нажатии на кнопку "Забронировать" пользователя перенаплавляет на страницу с формой отправки заявки.
Авторизованный пользователь может увидеть все свои отправленные заявки в личном кабинете.

Для того чтобы зарегистрироватся пользователь переходит на страницу регистрации через меню. На данной странице заполняет форму регистрации.
После успешной регистрации пользователь переходит на страницу авторизации и выполняет вход в аккаунт. Пользователя перенаправляет в его личный кабинет.

Перейдя на вкладку "Онлайн продление книги" пользователь может продлить книгу, взятую ранее. Авторизованный пользователь увидит свои заявки в личном кабинете после их отправки.

Перейдя на вкладку "Новости" пользователь может узнать какие мероприятия проходят в библиотеке.
При нажатии на картрочку с новостью пользователя перенаправляет на страницу с подробным описанием новости, где авторизованный пользователь может оставить комментарий.
Комментарии отобразятся на странице после того, как их одобрит администратор сайта.


Возможности админа:

Панель админа доступна под логином admin и паролем admin11

В админ панели есть возможности редактирования статуса заявок и комментариев, добавление книг, авторов.
